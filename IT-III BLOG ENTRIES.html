I - What is Computing?

Computing involves the use of computers and computational techniques for processing, analyzing, storing, and communicating information. This encompasses activities such as programming, data analysis, software development, and networking, playing a crucial role in diverse fields and shaping how we work, communicate, and solve problems.I've also learned about programming languages, web development, databases, cybersecurity, providing information and assistance in these areas.

II - Professions and Careers in Computing

I've learned that there are many professions available after completing a Bachelor of Science in Information Technology degree. The field of computing offers a variety of careers. There are software developers who shape the world system administrators who keep computer systems secure and data scientists who uncover insights, from vast datasets. Cybersecurity professionals in safeguarding assets against increasing cyber threats.Furthermore, these professions will serve as a source of motivation for you to pursue a BSIT course, inspiring you to contribute meaningfully to the ever-evolving landscape of information technology

III - IV Information Processing Cycle & Computer Systems

The computer science field defines the Information Processing Cycle as a series of steps including input, processing, storage and output. It emphasizes the role of the CPU, in executing instructions. Emphasizes how this cycle is a continuous loop. Moreover comprehending computer systems involves acknowledging how hardware components like the CPU and memory work together with software elements such, as operating systems and applications. This highlights the interplay required for data processing.

V. Concept of Computer Security

I've learned that Computer security, is centered on safeguarding computer systems, networks, and data from unauthorized access and damage. It ensures confidentiality through measures like encryption,and emphasizes availability by implementing robust backup systems and response plans. Addressing vulnerabilities through software updates, patches, and security audits, along with user education, is crucial for a comprehensive approach to maintaining the security of computer systems.

A LESSON THAT CHALLENGED ME THE MOST

Learning HTML and CSS can be challenging for beginners like me, as these languages form the foundation of web development. Understanding the syntax and structure of HTML for content and CSS for presentation can be daunting. Abstract concepts like styling, positioning, and the box model add complexity. The immediate visual feedback in browsers is rewarding but challenging. Staying updated on evolving standards complicates matters. Ultimately, achieving proficiency requires a nuanced understanding, practice, and hands-on experience to create well-designed, responsive web pages. Despite the challenges, I enjoyed learning, especially when I see the fruits of my hard work and the output of my code.
